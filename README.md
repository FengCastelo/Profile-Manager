# Profile-Manager
Sistema simples de login feito com HTML, Tailwind CDN e JavaScript. Inclui autenticação, validação de campos, avisos de erro e acesso a um painel após o login. O design é responsivo e minimalista, utilizando a paleta índigo do Tailwind. Projeto criado para praticar DOM, formulários e boas práticas de front-end.

#ENG VERSION
Simple login system built with HTML, Tailwind CDN, and JavaScript. Includes authentication, field validation, error feedback, and access to a panel after login. Features a responsive, minimalist design using Tailwind’s indigo palette. Created to practice DOM handling, forms, and front-end best practices.
